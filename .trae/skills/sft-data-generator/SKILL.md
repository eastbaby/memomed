## 技能概述

本技能旨在帮助用户快速生成高质量的SFT（监督微调）训练数据，支持明确规定场景、basemodel格式和用途，仅输出JSONL格式，同时提供便捷的方式让用户修改和优化AI生成的初始数据。

## 适用场景

- 为语言模型创建监督微调训练数据
- 构建对话系统的训练语料
- 生成特定领域的问答对
- 创建指令遵循任务的训练数据
- 其他需要高质量人工标注数据的场景

## 工作流程

1. **接收用户输入**：用户提供场景描述、basemodel格式要求、数据用途和生成数量等信息
2. **生成初始数据**：根据用户提供的场景、格式要求和用途，生成符合JSONL格式的初始训练数据
3. **展示生成结果**：以用户友好的方式展示生成的数据
4. **用户修改**：允许用户查看和修改生成的数据内容
5. **导出数据**：将最终数据导出为JSONL格式

## 支持的输出格式

- **JSONL**：每行一个JSON对象，适合大多数微调框架（唯一支持的格式）

## 生成数据的结构

### 对话格式（默认）

```json
{
  "messages": [
    {
      "role": "user",
      "content": "用户问题或指令"
    },
    {
      "role": "assistant",
      "content": "助手回复"
    }
  ]
}
```

### 指令格式

```json
{
  "instruction": "任务指令",
  "input": "可选的输入内容",
  "output": "期望的输出结果"
}
```

## 使用指南

### 基本使用

1. **提供场景描述**：详细描述你想要生成的训练数据的场景，例如：
   - "生成关于旅游景点推荐的对话数据"
   - "创建关于数学问题解答的指令数据"
   - "生成医疗健康咨询的问答对"

2. **指定basemodel格式**：明确规定assistant返回content的basemodel格式，例如：
   - "使用对话格式，包含role和content字段"
   - "使用指令格式，包含instruction、input和output字段"

3. **说明数据用途**：明确说明训练数据的用途，例如：
   - "用于微调语言模型，提高对话能力"
   - "用于训练特定领域的问答系统"
   - "用于指令遵循任务的模型训练"

4. **设置生成数量**：指定你需要生成的数据条数

5. **提供其他要求**：如有特殊要求，如语言风格、难度级别等，请一并说明

### 高级选项

- **数据多样性**：通过提供多个场景或角度，增加数据的多样性
- **领域特定**：提供领域知识或背景信息，使生成的数据更符合特定领域
- **格式定制**：如果需要自定义数据结构，请提供详细的格式说明
